# CL-6 — Implementation team is agentic, not human: CoderC (Claude Code orchestrator) + C

No human payroll. CoderC = Claude Code, which can spawn the Viabe Teams skill consisting of Pipeline Engineer (backend), Frontend Engineer (frontend), Quality Engineer (QC), Test Engineer (post-implementation tests) as sub-agents. CoderX = external code reviewer (mandatory on every PR; gates merges). Fazal = CEO, sole human, owns: Type 3 commitments, regulatory paperwork (KYC + DLT + DNS), Hindi translation review on every template, final security/privacy/legal sign-off, design partner onboarding in concierge mode. Implication: parallelism is unconstrained by hiring; bottleneck is Fazal's review bandwidth (20 hrs/week). 4-5 milestone-sized chunks per week is the realistic review throughput.

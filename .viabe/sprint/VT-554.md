# VT-554 — B3 action-path + config-gated enforce

## Built (0 migrations)
- **Uniform action-return envelope** — `recommend_sales_play` now emits `pushback=False` + `action_taken` + `outcome` alongside its existing intent fields (backward-compatible), so an ACTION flows through the SAME `specialist_return` bridge as a pushback. The manager decision loop now observes ACTIONS (ACCEPT / NEXT_SPECIALIST), not just pushbacks.
- **`handle_specialist_return(env, *, agent, enforce=None)`** — observes the decision (always, as before) AND, when ENFORCING, acts on the high-stakes ESCALATE deterministically: opens + VTR-escalates an incident (VT-552 store, tier 2) rather than leaving an in-lane-infeasible, no-path pushback to the model. `enforce` defaults to the **`MANAGER_ENFORCE_ROUTING` env flag (default False)** → today byte-for-byte the observe-only path (CL-2026-07-01-observe-only-rails). The sales + finance pushback wires + the new action wire all route through it.

## Why config-gated default-off
The routing-steer flip touches the live agent loop — per the observe-only-rails discipline it lands as a rail (built + tested with `enforce=True`) but ships OFF until validated on dev. Flipping `MANAGER_ENFORCE_ROUTING=true` is the separate, deliberate step (like OC1's `enforce_policy`).

## Verify
ruff clean; dep-less-safe (incident_store lazy-imported only on enforce); **48/48** — action envelope observed → ACCEPT; `enforce=True` + no-path ESCALATE → `create_incident` + `escalate_incident(to_tier=2)` fire; `enforce=False` (default) → no incident (observe-only byte-for-byte); env flag read; + the pre-existing bridge / sales_lane / finance / decision tests (finance test repointed to `handle_specialist_return`).

## NOT in scope
The actual `MANAGER_ENFORCE_ROUTING=true` flip (validated follow-up). Enforcing the non-ESCALATE decisions (REVISE re-dispatch / CLARIFY ask-owner) — this slice enforces only the clearest high-stakes case (ESCALATE); the rest stay advisory. Action wires for the other lanes (only sales has a live action tool today).

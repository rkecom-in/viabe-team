# CL-130 — VT-3.4 spec uses outdated langgraph-supervisor API

VT-27 spec (written 2026-05-15) shows:

  supervisor_workflow = create_supervisor(
      agents=[orchestrator_agent, sales_recovery_agent],
      model=opus_4_7,
      prompt=ORCHESTRATOR_AGENT_SYSTEM_PROMPT,
      handoff_tools=[spawn_sales_recovery]
  )

Current langgraph-supervisor API per Context7 2026-05-18:

  supervisor_workflow = create_supervisor(
      [orchestrator_agent, sales_recovery_agent],  # positional
      tools=[spawn_sales_recovery],  # NOT handoff_tools
      model=opus_4_7,
      prompt=ORCHESTRATOR_AGENT_SYSTEM_PROMPT,
  )

Briefs will use the current API. Notion VT-27 spec page should be updated post-merge to match shipped code; deferring the doc update since the code is authoritative once merged.

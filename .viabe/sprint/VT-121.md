## Concept reference
- Joint launch tracker (`Viabe_Launch_Tracker` Notion database, data source `413be4ab-870d-4895-bf35-dfd579142001`) is the calendar-anchored source of truth for 45 milestones across Reports-Jun15, Team-Jul15-Soft, Team-Aug15-Full launch gates.
- `concept-team.md` §15 (observability surfaces Fazal needs daily for solo-founder operations).
- `concept-team-pillars.md` Pillar 7 (operational visibility cannot be silently degraded).
## Why this subtask exists
Fazal has a 95-day window across three launch gates with 45 milestones, ~13 of which Fazal personally owns. Manually checking the Launch Tracker every morning is the gap that will get missed first. This subtask delivers a deterministic daily-alert mechanism: every weekday at 9 AM IST, scan the Launch Tracker for overdue + due-soon milestones and post a structured Telegram message to Fazal. Missed milestones become impossible to silently ignore.

## What CoderC must produce
### 1. GitHub Action workflow at `.github/workflows/launch-tracker-daily-alert.yml`
- Schedule: cron `30 3 * * 1-5` (9:00 AM IST = 3:30 UTC; weekdays only Mon-Fri; weekends excluded).

- Steps: checkout, setup Python 3.12, install deps (`requests`, `python-dotenv`), run `scripts/launch_tracker_daily_alert.py`.
### 2. The alert script at `scripts/launch_tracker_daily_alert.py`
Deterministic Python script. No LLM calls. ~150 lines.
Flow:
1. Query Notion API: `POST /v1/databases/413be4ab-870d-4895-bf35-dfd579142001/query` with filter `{Status NOT IN [Completed, Deferred]} AND {Target Date < today}`.
2. Parse response into list of overdue milestones (name, target_date, owner, launch_gate, track, linked_vt_tasks, blocking_notes).
3. Sort by days_overdue descending.
4. Build Telegram message: header + OVERDUE section + DUE THIS WEEK (next 7 days) section + 'ALL CLEAR' fallback if both empty.
5. POST to Telegram via `https://api.telegram.org/bot{TELEGRAM_BOT_TOKEN}/sendMessage` with `chat_id={FAZAL_TELEGRAM_CHAT_ID}` (split into multiple messages if >4096 chars).
6. Log output: count of overdue, count of due-soon, success/failure.
7. On Notion API failure: post SEPARATE Telegram failure-alert. Never silently fail.
### 3. Telegram message format example

```javascript
LAUNCH TRACKER DAILY DIGEST  2026-05-19 (Tuesday)

OVERDUE (2):
  Fazal     | Reports-Jun15    | Razorpay Live KYC submitted              | 4d overdue

DUE THIS WEEK (3):
  Fazal     | Team-Jul15-Soft  | Meta Tier-A templates authored           | due in 0d

  Claude C  | Reports-Jun15    | Viabe Router Vercel project created      | due in 3d

Full tracker: https://notion.so/544492b9484b457b8415630e0362bfda
```

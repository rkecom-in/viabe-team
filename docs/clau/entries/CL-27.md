# CL-27 — VT-3 Orchestrator parent and VT-24 through VT-31 subtasks need respeccing for or

VT-24 (LangGraph skeleton) becomes 'orchestrator-agent runtime ON LangGraph substrate' rather than 'LangGraph state machine'. VT-25 (SubscriberState) shifts from owning routing logic to providing the orchestrator-agent's memory hooks. VT-26 (Twilio webhook receiver) unchanged. VT-27 (deterministic pre-filter; previously the diamond intent classifier) becomes Stage 1 of two-stage filtering, NOT an LLM call. VT-28 (orchestrator dispatcher) becomes the orchestrator-agent's reasoning loop with spawn_specialist tool. VT-29 (scheduled triggers) unchanged. VT-30 (error framework) extends to handle orchestrator agent self-errors. VT-31 (output composer) becomes one of the orchestrator-agent's tools. New subtask needed: VT-3.N for L0 memory tier integration. New subtask needed: VT-3.N+1 for orchestrator system prompt + tool inventory.

# CL-272 — TECH DEBT / NEXT ACTION: self_evaluate gate is inert post-VT-50 — sales_recovery

It is a MISSING SUBTASK: 'wire SelfEvaluateAdapter into sales_recovery_node'. The 2026-05-12 respec did not have this row; CL-266's canonical sequence did not surface it because the wiring need only became visible once VT-36's evaluator=None default met VT-50's adapter. IMPACT ON SEQUENCE: VT-4 (SalesRecovery-Agent) Definition of Done requires a canary run that 'passes self-evaluate' — that DoD is NOT met by VT-50 merging; the wiring step must land first.

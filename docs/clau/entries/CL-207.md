=== OBSERVED MECHANISM ===
Not strict short-circuit. Actual mechanism is convergence + langgraph frontier-dedup. On the spawn path:
1. orchestrator_agent's create_agent subgraph emits tool_call for spawn_sales_recovery.
2. spawn_sales_recovery tool returns Command(goto='sales_recovery_agent', graph=PARENT).
3. route_after_orchestrator ALSO fires on the conditional edge, reads tool_calls, returns 'spawn'.
4. Conditional-edge mapping {'spawn': 'sales_recovery_agent', 'terminal': 'orchestrator_terminal'} resolves 'spawn' → sales_recovery_agent.
5. Command.PARENT goto and conditional-edge destination converge on sales_recovery_agent.
6. langgraph frontier is a set → sales_recovery_agent named twice = executed once.
7. orchestrator_terminal never enters frontier on spawn path.

On the no-spawn path: Command.PARENT does not fire (no tool_call). Conditional edge does all routing work — route_after_orchestrator returns 'terminal' → orchestrator_terminal node executes, sets terminated_without_spawn=True.

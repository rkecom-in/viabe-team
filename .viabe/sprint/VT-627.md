# VT-627 — two dispatch-brain prompt fixes surfaced by the honest gate measurement

`orchestrator_agent_system.md` (the dispatch/handle-directly brain).

## Verbosity clamp (gate lever #3)
VT-625 (raising the 1000-char log cap) exposed that the manager sends WALLS of text — happy_path's full reply
was 2098 chars, judge dinged progression (a wall the owner won't read). The dispatch brain prompt said only
"concise" (line 16) with NO explicit length rule, unlike the journey brain (`turn_brain.py:92` "keep every reply
SHORT — one or two sentences"). **Fix:** added a hard rule to the reply-writing paragraph — "Keep every reply
SHORT — one or two WhatsApp-style sentences; no walls of text / markdown dumps / essays; longer ONLY when the
owner explicitly asked for detail. A concise reply that moves forward beats a thorough one the owner won't read."

## Hinglish mirror (gate lever #4a)
The dispatch brain's only language cue was the soft word "in-language" (line 16); the journey brain has the
explicit "if they write in Hindi/Hinglish, reply in Hinglish". food_platform_hinglish (on sonnet) replied in
ENGLISH to a 100% Hinglish owner. **Fix:** replaced "in-language" with an explicit hard rule — "Mirror the
owner's language and register — if they write in Hindi or Hinglish (romanized Hindi, e.g. 'kaise jodu isse'),
reply in Hinglish; never default to English when the owner wrote in Hinglish."

## Hinglish TIER (gate lever #4b) — DEFERRED, Fazal's cost call
The root-cause found haiku-4-5 FABRICATES on Hinglish "question" turns (`dispatch.py select_brain_model` routes
question/status_query → haiku). The prompt ALREADY forbids fabrication → this is a model-capability gap the mirror
fix above does NOT close; the real fix is routing Hinglish off haiku → sonnet. That is a **COGS increase (~10x on
those turns)** Fazal explicitly reserved as his call ([[vt619-cost-metering-and-pricing]]), and it's only
MED-confidence (2 Hinglish scenarios failed even on sonnet). NOT unilaterally flipped — surfaced for Fazal to
weigh cost vs quality. A deterministic romanized-Hinglish detector (function-word wordlist, like VT-626's
verb detector) would gate it if he wants it. See [[vt624-625-gate-honesty-and-levers]].

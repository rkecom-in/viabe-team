## 2. The autonomy-first principle (Fazal, near-verbatim)

*"We need to be Autonomous, we can't keep asking the owner for every approval. The
objective of Viabe Team is to work as an independent agent. As long as the agent is not
suspected to be causing monetary or reputational damage, and the action the agent is
taking is logical, the agent can just continue without questioning the owner repeatedly.
We are creating something that is expected to run the business on behalf of the owner.
Just like Claude Code, we can have permission options like Ask and Act — Ask involves
the owner on every necessary action; Act executes without asking."*

# VT-553 — Track D advice-quality eval harness

## What (per CL-2026-07-01-no-fixed-playbook)
Knowledge = the LLM's reasoning + the C3 learnable memory, not an authored note-set. This harness MEASURES that output before a capability graduates — it never authors/scripts/confines advice and is NOT a retrieval corpus.

## Built (0 migrations, pure + dep-less)
- **`advice_eval.py`** — `EvalCase` (scenario + grounding context; **no expected_answer** — measuring, not comparing to a target), `EvalDimension` (factuality/actionability/relevance/tone), `score_advice`/`run_eval`/`EvalReport` (per-dimension aggregate + `pass_rate(threshold)`).
- **The no-fabricated-numbers RAIL** (`find_fabricated_numbers`, deterministic) — the ONE surviving guardrail: any significant numeric claim (%, currency, ≥2-digit) in the advice NOT grounded in the case context is a factuality **hard-fail to 0.0**, independent of the softer LLM judge. Single-digit structural numbers ("3 tips") are not flagged.
- **Pluggable judge** — `LLMJudge` (real, lazy-imports the model → runs on deployed dev where the key lives) + `StubJudge` (deterministic, for tests/dry-runs). The harness is dep-less-importable + unit-testable with the stub.
- **`advice_eval_cases.py`** — a STARTER held-out measurement set (3 cases), clearly marked measurement-only / never-read-by-agent. The full set is a Fazal/archetype follow-up.

## Verify
ruff clean; **dep-less-safe** — 8/8 collect + pass under the strict `--isolated` smoke (the env that caught the VT-552 trap), no model/DB: fabricated % flagged when ungrounded / grounded numbers + single digits not flagged / overall = mean dimension / **fabrication hard-fails to 0.0 even with a 0.95 judge** / `run_eval` aggregates + pass_rate / held-out cases load + carry no expected_answer.

## NOT in scope
The real LLM-judge scoring RUN (LLM-dependent → deployed dev, per validate-on-dev). The full held-out set CONTENT (Fazal/archetype follow-up). Wiring the harness into a graduation gate (C4 thresholds — Fazal-approved before first graduation).

# VT-628 — the manager AND its ruler both sampled at temperature 1.0

The stocktake found the top gate blocker is not (only) the model's intelligence — it's that **intelligence
was being measured with a rubber ruler**. `temperature` was set on ZERO LLM calls in the orchestrator → both
the manager brain and the eval judge sampled at the API default 1.0. So the ±1-2 per-scenario swing ("variance")
is partly the manager being non-deterministic and partly the JUDGE scoring an identical transcript differently
run-to-run. Cannot chase a quality floor with a moving ruler.

## Family gate (verified live, both directions)
`claude-opus-4-7` AND `claude-opus-4-8` **400 on the `temperature` param** (tested against the live API this
session). haiku-4-5 / sonnet-5 accept it. New `src/orchestrator/llm_config.py::sampling_kwargs(model_id)` →
`{}` for opus, `{"temperature": 0.0}` otherwise.

## What got pinned (non-opus → temperature=0)
- **Brain** — `dispatch.py::_resolve_model` (haiku/sonnet, every turn — the biggest lever; also covers the
  VT-593/594 compose fallbacks, which route through `_resolve_model`).
- **Intent classifier** — `classify_owner_message.py` (haiku). Kills the tier-flip non-determinism: the
  haiku-vs-sonnet brain choice is a pure lookup on this signal, so a deterministic classifier → deterministic tier.
- **Triage** — `manager/triage.py` (sonnet).
- **Review/extraction** — `manager/review.py` (sonnet).

## What CANNOT be pinned (documented limit)
- **The 10 opus specialist lanes** (sales/marketing/finance/tech/cost/accounting/onboarding-conductor/
  integration/orchestrator-agent + field_mapping) are `claude-opus-4-7/4-8` → reject temperature → **irreducible
  temp-1.0 variance** on specialist CONTENT (e.g. SR campaign wording). Pinning them needs a model change, not a
  param — folds into stocktake step 6 (reasoning-tier decision, Fazal's cost call). The brain authors the
  majority (handle-directly) path, so this is the smaller slice.

## The ruler (eval judge)
`transcript_judge.py` DEFAULT_MODEL `claude-opus-4-8` → **`claude-sonnet-5` + temperature=0** (family-gated in
the call). opus can't be made deterministic; a fixed sonnet ruler beats a sharper wobbling opus one for a
go/no-go floor. **Rebaselines the number** — the old ~43% was opus-at-temp-1.0 and is NOT comparable.

## Target (Fazal 2026-07-08 — reframed from a flat 95%)
"A manager not on par to run my business, I won't trust." → the bar is a TRUST/ACCEPTANCE bar, two-tier:
- **Trust-breakers = 0 (100% clean):** fabrication, wrong/dropped money action, verbatim-repeat loop, ignoring
  the speech-act asked, promising the impossible. Measured as a violation COUNT, not an average.
- **Quality acceptance ≥ 90%** to ship as trustworthy (95% = excellent). Not a stochastic 95% average on a
  rubric that scores a proactive campaign-draft as "overreach".

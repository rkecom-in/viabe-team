# team-shared/db

Future home for **generated database types** for the shared `viabe-team-prod`
Postgres schema.

Empty for now. Populated when the first real migration lands in
**VT-Foundation** — types are generated from the schema defined in the
repo-root [`/migrations/`](../../../migrations/) directory and consumed by
all three apps.

Do not hand-write types here; they are generated.

# CL-23 — Mermaid diagram styling pattern: stroke-width:2px mandatory, explicit color:#000

When writing Mermaid diagrams for Notion, the styling pattern from the Pipeline Observability diagram is the canonical template: `classDef <name> fill:<lighter-pastel>,stroke:<darker>,stroke-width:2px,color:#000;` with explicit semicolons. Three rules: (1) stroke-width:2px is mandatory — hairline default strokes disappear against pastel fills at any zoom level, killing the color-coding meaning. (2) color:#000 is explicit — defensive against renderer themes overriding to white text. (3) Lighter fills + slightly darker strokes (e.g. fill:#f1d6ff with stroke:#7700cc) read better at small sizes than uniform medium-saturation pastels. Applied to the Pipeline Observability diagram on 2026-05-12 per Fazal's improved styling.

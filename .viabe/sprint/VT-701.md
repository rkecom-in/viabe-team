## What shipped

- **Turn-brain prompt, 3 new rules**: PLAIN WORDS (everyday language, no "operate"-style
  jargon); NEVER DEFLECT CONFUSION (a "what does that mean?" gets a one-line explanation with
  a concrete example + the same buttons re-offered); STAY ON THE OBJECTIVE (the asked question
  MUST be the current still-needed field — rephrase allowed, topic substitution forbidden;
  extractions recorded ONLY under the field actually asked).
- **Per-question help**: `Question.help_en/hi` — gap LLM now returns a plain one-line
  explanation per question (prompt updated, PLAIN-WORDS rule added there too);
  `_FIELD_HELP` deterministic fallback for the canonical fields (finite enum);
  `field_help()` exported; help rides `_compose_queue` dicts and renders into the turn-brain's
  still-needed context ("meaning: …") so explanations are ACCURATE, not improvised.
- **Represent guard upgraded** (runner `_journey_represent_instead_of_consent_ask`): the
  re-present now leads with the question's plain-language help and rides `journey._send`
  (suggestion buttons / card formatting come along). Deterministic — no LLM on this path.
- **JOURNEY-GATE-FAIL-OPEN grep marker** on the journey gate's outer catch — every firing is
  a real defect to chase (run #4's firing was invisible in the bounded log window).

# VT-589 — the manager's answer actually reaches the owner

## Root cause (opus trace, file:line verified)

## Fix (smallest correct, preserves the no-send-tool invariant)

That records an assistant turn → _brain_emitted_owner_reply True → the D1 fallback correctly stops (no double-send). Best-effort: a send failure leaves the D1 fallback as the net. The send lives in the deterministic dispatch seam, NOT the agent tool inventory — tool_guardrail invariant intact.
